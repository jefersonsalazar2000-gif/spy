# Stock Analyzer Pro — JEFER85
## Instalación en 3 pasos

### Requisitos
- Node.js instalado (https://nodejs.org)

### Pasos

**1. Abrir terminal en la carpeta del proyecto**

**2. Instalar dependencias:**
```
npm install
```

**3. Iniciar el servidor:**
```
npm start
```

**4. Abrir en el navegador:**
```
http://localhost:3000
```

### Listo ✅
El dashboard carga con datos reales de Yahoo Finance + análisis IA de Claude.

- Las 7 Magnificas: AAPL, MSFT, GOOGL, AMZN, NVDA, META, TSLA
- SPY (S&P 500 ETF) y Ford (F)
- Agrega cualquier ticker en el campo de texto
- Auto-refresh cada 10 minutos
